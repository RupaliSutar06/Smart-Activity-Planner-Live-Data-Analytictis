# 🌐 Smart Activity Planner using Live Weather Data Analytics

---

## 🚀 Project Overview
Smart Activity Planner is a **Full Stack + Real-Time Data Analytics Project** that provides intelligent activity recommendations based on live weather conditions.

This project uses live weather data to suggest suitable activities depending on environmental conditions like temperature, AQI, and weather status.

---

## ⚡ Real-Time Data
This project uses **OpenWeather API** to fetch live weather data, making it a real-time application.
